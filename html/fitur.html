Fitur MauBNC


Mudah digunakan

Karena MauBNC menggunakan platform ZNC, Bouncer dari MauBNC sangat mudah di gunakan oleh orang baru kenal irc sekalipun.

1 Akun 2 Jaringan

Salah satu kelebihan MauBNC adalah 1 akun bisa dibuat 2 jaringan berbeda, dan semua aktif, ini seperti kamu memiliki 2 bouncer.

Tambah/Hapus Network

Ingin menambah dan menghapus jaringan terserah kamu, asal jangan menambah ke jaringan yang ada di Daftar Jaringan Hitam.

Tambah/Hapus Server

Menambah dan menghapus server pada jaringan kamu pun bisa kamu lakukan sendiri, tanpa bantuan admin.

Mendukung Server IPv6

Selain mendukung Oidentd, server MauBNC juga mempunyai alamat IPv6 :/64, ini bisa di jumlah lebih dari 1 miliar IP.

Bisa Ganti vHost sendiri

Bosen dengan vhost, kamu bisa ganti ke lainya, cukup lihat daftar bindhost yang tersedia, saat ini lebih dari 50+ vHost bisa digunakan.

Buffering Sampai 500+ Line

Kamu tak perlu takut kehilangan momen irc, karena selama kamu offline, semua kejadian akan direkam dan diserahkan kamu jika kamu sudah online.

Ada modul Aktif/Mati kan warna

Takut lag atau progam tidak merespon ketika ada flood / memainkan rekaman? matikan warna, kamu akan bebas dari lag, ini juga bisa mengirit kuota kamu.

Module paling lengkap dan terupdate

Tak usah khawatir, semua modul sudah ada dan bisa digunakan tanpa batasan di MauBNC, module baru sekalipun pasti ada.

ZNC Selalu di update secara berkala

Tak usah takut ketinggalan canggih, karena crontab akan update melalui git otomatis secara berkala, so ZNC kamu adalah ZNC paling baru di versi sekarang.
