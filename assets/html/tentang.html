MauBNC adalah proyek aktif BNC gratis yang dibuat dari platform ZNC, MauBNC adalah merk penyedia Bouncer Gratis menggunakan ZNC yang sudah dimodifikasi, MauBNC mempunyai server tunggal yang handal dan sangat cepat, Server MauBNC adalah server dedikat milik pribadi, namun dibagi untuk semua, dan proyek ini adalah proyek perorangan, di MauBNC tidak ada kami, yang ada hanyalah saya, saya adalah chatter IRC, terhitung mulai tahun 2008 sampai sekarang saya masih gemar yang namanya chat di IRC Karna saya memiliki server pribadi dan saya tahu bagaimana saya membuat ZNC, maka terlahirlah MauBNC ini, MauBNC hanya didirikan oleh satu orang, dan orang itu adalah saya, saya sendiri membuat proyek MauBNC tidak untuk tujuan lain selain

Berbagi dengan Kamu apa yang saya punya saat ini
